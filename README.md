# Kursovaya_elevator

HTML-макет веб-приложения для имитации автоматизированной системы контроля работы лифтов.

## Описание

Интерактивный макет демонстрирует пользовательский интерфейс будущего веб-приложения по теме курсовой работы «Лифты». Реализован в тёмной теме без зависимостей: чистый HTML + CSS + JavaScript.

## Запуск

Открыть файл `index.html` в любом современном браузере.

## Структура

- `index.html` — разметка интерфейса.
- `styles.css` — стили (тёмная тема).
- `app.js` — логика макета: моделирование шахты, кнопок вызова, статистики.
